Mac Tip's

naijoug
2016-01-12
Mac

收集整理个人使用Mac的一些小技巧。

Mac显示隐藏文件

在Mac上显示和隐藏文件必须通过命令行进行操作。需要注意的是通过命令行输入后，需要重启一下Finder才能生效。

Mac截图

在Mac上截图有很多中方法，都包括截图的基本功能（窗口截图、屏幕截图、选择截图）。

系统自带小工具：抓图

  • 用法 : 可以通过Spotlight搜索抓图即可打开。比较特别的是包括一个倒计时10秒截图的功能，
  • 缺点 : 需要打开软件才可以进行截图操作，没有全局的快捷键，所以平时使用比较少。

QQ自带的截图功能

  • 用法 : QQ的截图的必要条件当然时必须打开QQ，当然一般情况下QQ都是登录状态的，所以使用还是比较方便的。
  • 缺点 : 在无网络情况下不能登录QQ，就无法使用。

快捷键截图：Command + Shift + 4

  • 用法 : 系统自带的快捷键截图，通过空格键可以进行窗口模式和自定义区域模式切换，截图完成之后自动保存在桌面。(快捷键Command + Shift + Control + 4将自动保存在剪切板。)
  • 缺点 : 没有取消功能，截图完成之后就直接保存到桌面或剪切板。

设置Mac的默认打开程序

Mac设置默认打开程序的方法如下：选中要打开文件 -> 右键 -> 显示简介 -> 打开方式

Mac文本编辑器”直引号”会变“弯引号”

Word、Pages、备忘录，使用英文引号""(直引号)，可是打出却变为了“”(弯引号)。这几个文本编辑软件都默认打开智能替换的功能，手动进行关闭就行了。

  • Word : 偏好设置 -> 自动更正 -> 键入时自动套用格式 -> "直引号"替换为“弯引号”(去掉选择对号)
  • Pages、备忘录 : 编辑 -> 替换 -> 智能引号(去掉选择对号)

Apple提供的编辑软件Pages和备忘录，都是默认开启了智能替换功能，包括：智能引号、智能破折号、智能链接…

  • 智能引号 : 就是将""(直引号)转化为“”(弯引号)
  • 智能破折号 : 当输入多个破折号-时，会将多个破折号连成一条长破折号
  • 智能链接 : 把网址复制到文本中时，会自动转化为超链接
